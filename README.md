# Сравнительный анализ языковых характеристик прозы Чехова и Зощенко методами корпусной лингвистики

> *Методы Digital Humanities в исследовании структуры и языка художественного текста*

**Финальный проект** по курсу «Программирование для цифровых гуманитарных проектов» НИУ ВШЭ, магистерская программа «Цифровые методы в гуманитарных науках»

___

Данное исследование посвящено сравнительному анализу идиостилей А. П. Чехова и М. М. Зощенко на материале сбалансированного корпуса из 50 рассказов.

**Цель работы** — провести комплексный количественный анализ языковых характеристик малой прозы двух классиков и проверить гипотезу о статистически значимых различиях их стилей.
